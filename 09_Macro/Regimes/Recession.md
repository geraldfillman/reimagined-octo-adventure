## Definition
Two consecutive quarters of negative real GDP growth, or NBER official declaration. Characterized by rising unemployment, falling corporate earnings, credit tightening, and risk-off sentiment.

## Indicator Thresholds
| Indicator | Trigger Level | Current | Signal |
|-----------|--------------|---------|--------|
| [[GDP Growth]] | <0% for 2 quarters | | |
| [[Unemployment Rate]] | Rising >0.5% from trough | | |
| [[PMI]] | <47 sustained | | |
| [[Yield Curve]] | Uninversion after inversion | | |
| [[Initial Claims]] | >300k sustained | | |

## Sector Impact Map
### Favored (Defensive)
- [[Consumer Staples]] — inelastic demand
- [[Utilities]] — regulated revenue, dividend yield
- [[Healthcare]] — non-discretionary spending

### Challenged (Cyclical)
- [[Consumer Discretionary]] — spending pullback
- [[Industrials]] — capex freeze
- [[Financials]] — loan losses rise

## Historical Precedents
- 2020: COVID recession (shortest ever)
- 2007-2009: Great Financial Crisis
- 2001: Dot-com bust
